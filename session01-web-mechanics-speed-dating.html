Introduction aux mécaniques du Web

CC-BY-SA par Mozilla

20 minutes

Les mécaniques du Web sont un ensemble d'outils, de caractéristiques et de fonctions qui définissent comment nous utilisons internet. Comprendre ces mécaniques nous permet de protéger nos informations personnelles en ligne, d'utiliser et de comprendre les URL, les adresses IP, les termes de recherches, les marque-pages et plus encore. Cette activité présente les bases des mécaniques du Web via une activité hors-ligne pendant laquelle interagissent plusieurs personnes. Cette activité peut être adaptée pour tous les âges.

Objectifs d'apprentissage

  • Connaître et comprendre le vocabulaire relatif aux mécaniques du Web pour la navigation en ligne et la protection des informations personnelles.
  • Construire une communauté avec les personnes qui participent à l'évènement.

Public

  • Préparation

    Créer des cartes : Créez des cartes avec des noms pour chacune des mécaniques du Web listées ci-après. Chaque carte a deux faces. Sur le recto, on écrira le nom de la mécanique (par exemple « Cookies »). Sur le verso, on écrira la description de cette mécanique en quelques phrases (par exemple « je suis un morceau de texte envoyé par un site web à votre navigateur »). Vous pouvez réaliser ces cartes avant le début de l'activité ou demander aux participants de contribuer à leur réalisation.

    • Cookies : Je suis un morceau de texte envoyé par un site web à votre navigateur.
    • Bande passante : Je suis une voie électronique qui connecte votre ordinateur à Internet. Si on m'augmente, la connexion Internet sera plus rapide.
    • Marque-pages : Sur le Web, je suis un identifiant qu'on stocke pour accéder à une page plus tard.
    • Langages de programmation du Web : J'indique aux navigateurs web comment structurer et présenter le contenu d'une page web.
    • URL : Je suis une chaîne de caractères, un texte, qui est une référence vers une ressource.
    • Paramètres de vie privée : Je permets aux gens qui utilisent le Web de définir la façon dont les sites peuvent accéder à leurs informations.
    • I.P. : Je suis une étiquette numérique qui est associée à chaque appareil d'un réseau informatique. Mes deux lettres signifient « Internet Protocol » (pour « protocole Internet »).
    • Navigateur : Je suis un logiciel qui permet de récupérer des ressources sur le Web, de les consulter et d'envoyer des informations sur le Web.

    Participants : En considérant que les participants découvrent les mécaniques du Web, trouvez une façon de présenter ces concepts. Cela peut s'inscrire dans une discussion, sous la forme d'une leçon, de questions-réponses, etc. Les participants devront ensuite jouer le rôle de la mécanique décrite sur leur carte et réagir comme s'ils rencontraient d'autres mécaniques du Web. De cette façon, les participants pourront s'enseigner les mécaniques du Web lorsqu'ils se présenteront. L'objectif est que chacun soit à l'aise, une fois que l'activité commence, pour discuter des différentes briques qui composent les mécaniques du Web.

  • Activité

    15 minutes

    Présentation de l'activité : Regroupez les personnes qui participent à l'activité. Chaque personne reçoit une carte. Demandez aux participants de prendre le temps de lire le recto et le verso de leur carte. Ensuite, chacun fera semblant d'être la mécanique décrite sur leur carte. Une fois l'activité commencée, ils auront deux minutes pour apprendre à connaître les autres mécaniques présentes dans la salle.

    Commencer l'activité : Lancez un chronomètre pendant deux minutes. Dites aux participants de trouver une autre personne avec laquelle discuter (cela fonctionne mieux lorsqu'on a un nombre pair de participants, mais vous pouvez également participer s'il y a un nombre impair). Pendant ces deux minutes, les deux personnes doivent se présenter et se poser des questions comme celles qui suivent. Lorsque les deux minutes sont écoulées, chaque participant doit trouver quelqu'un d'autre et faire la même chose. Répétez cette étape jusqu'à ce que chacun ait pu rencontrer tous les participants.

    • Que faites-vous dans la vie ?
    • Comment sont vos amis ?
    • Comment pouvons-nous garder contact ?
    • Quel âge avez-vous ?
    • D'où venez-vous ?

  • Réflexions

    5 minutes

    Une fois l'activité terminée, regrouper les participants en cercle et leur demander d'échanger sur l'activité, de décrire les défis rencontrés, les moments amusants ou les choses qu'ils ont apprises.
